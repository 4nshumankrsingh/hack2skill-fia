## 📖 Overview

This project is a single-page futuristic hackathon landing page built with modern frontend technologies and premium UI/UX practices. The website focuses on immersive visuals, smooth animations, accessibility, responsiveness, and component-driven architecture.

The design combines:

- Dark futuristic aesthetics
- Glassmorphism UI panels
- Neon purple accents
- Smooth Framer Motion transitions
- Interactive sections and timelines
- Mobile-first responsive layouts

The project was created to demonstrate strong frontend engineering skills using React + TypeScript while maintaining production-quality structure and performance standards.

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js v18+
- npm v9+

---

### Installation

```bash
# Clone repository
git clone https://github.com/your-username/hackathon-project.git

# Navigate into project
cd hackathon-project

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will start on:

```bash
http://localhost:5173
```

## 🏗️ Production Build
